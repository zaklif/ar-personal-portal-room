# AR Personal Portal Room

An Android Augmented Reality (AR) application developed using Unity and AR Foundation that allows users to create customizable virtual rooms for storing, organizing, and visualizing 3D objects through an immersive portal experience.

## Project Overview

AR Personal Portal Room provides users with a virtual space where they can customize room environments and place 3D objects using AR technology. The system is designed for collectors who want to visualize and organize their collections digitally.

## Features

- AR portal room experience
- Customizable virtual rooms
- Room template selection
- 3D object placement
- Import GLB 3D models
- Object movement, rotation, and scaling
- Object metadata management
- Firebase authentication
- Room saving and loading
- Visitor and owner room access

## Technologies Used

- Unity 6
- C#
- AR Foundation
- Google ARCore
- Firebase Authentication
- Firebase Realtime Database
- Blender / GLB 3D Models
